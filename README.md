# E-commerce Complaint Chatbot

A NestJS-based chatbot that handles customer complaints regarding:
- Order status
- Refund issues
- Delivery delays
- Payment failures

## Tech Stack
- NestJS
- MongoDB
- LangChain
- Google Gemini API

## Features
- Polite and professional responses
- Conversation memory
- Persistent chat history# ecommerce-complaint-bot
